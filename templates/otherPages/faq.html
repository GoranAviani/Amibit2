Setup for Firefox on a computer


Install "New Tab Override" from Firefox Add-ons

Download


Select "New tab - settings" in the top right corner of your Firefox window to open settings.

In the settings window for OPTION select "custom URl" and for URL type in "www.amibit.org".

We recommend that you select Focus - "Set focus to the web page instead of address bar" which will set focus to our search.

Setup for Chrome on a computer


Install "Change New Tab" from hrome web store

Download


Select Add to Chrome button.

After instalation is done Change New Tab window will automatically open. Type in "www.amibit.org" and click X.

Done! When you select New Tab in Google Chrome it will take you to Amibit!

Setup for Firefox on mobile device


Select three dots in the top right corner of your Firefox window to open Firefox menu.

In the Firefox menu select "Settings" and then select "General".

In the General window select "Home" and then select "Set a Homepage".

In the new window select "custom" and type "www.amibit.org" as a address.
